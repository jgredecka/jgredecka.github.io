Projects

I have had the opportunity to undertake both individual and group projects throughout my course.


KSEAPlus

Flask app implementing algorithms for protein kinase activity inference.

Data visualised with matplotlib and seaborn.

Utilises Celery and Redis along with AJAX requests to run analyses in the background.

KinBase
(under construction)

Web app enabling users to search for data on protein kinases and related molecules.

Built with the microframework Flask.

SQLAlchemy used as an ORM to query and retrieve data stored in SQLite database.

KiNet

Python program that retrieves protein information from UniProt RESTful API.

Libraries used include urllib and urlib2.

Generated .SIF file is used to generate an interaction network via Cytoscape software.

PDBParser

Python program that parses and analyses protein structural data from PDB files.

Enables single and multiple file analysis.

matplotlib used as a data visualisation tool for barplot generation.
